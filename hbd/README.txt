Usage:
----------
a) Download and Unzip Jquery happy birthday script.

b) You can change text and flowers in this script.

c) Follow the instructions given in the read me text files.

Index.html:
-----------

This file contains the entire functionality of "Happy Birthday Script ".


Images Folder:
--------------

1) Images folder contains the images required for the script.

2) Replace the images in images folder with your own images.


jquery-1.9.1.min.js:
-------------------------

jquery-1.4.4.min.js file contains the jquery library functions that supports the jquery animations. Do not change the code in this file. If changes are done in this file, then it will affect the functionality of the script.


flytext.js:
---------

1. flytext.js file contains the text effect of the script.

2. The script will work only when the flytext.js file is included.

3. If you want to change the  text then you can change the below variables in flytext.js file.

             var message="Wish You.Many.More.Happy Returns.of the day ";
     
             

picture-fall.js:
---------


1. picture-fall.js file contains the image falling effect of the script.

2. The script will work only when the picture-fall.js file is included.

3. It gives you an awesome flowers falling effect with jquery.

4.Can change the wishes texts and flowers as per your needs.       

How to embed the script in to your webpage?
--------------------------------------------------------------
1) Open the index.html file.

2) Copy the script from index.html file, then replace the images and css properties according to your needs.

3) Copy and paste the script code inside the <body></body> tag of your webpage where you want to show the script.

4) After  replacing the images in images folder, rename the images as mentioned below.
 Actual Code:
<img src="./images/pause.png">
<img src="./images/play.png">



5) Make sure that the flytext.js, picture-fall.js, style.css and jquery-1.9.1.min.js and image files are placed in appropriate folder.
